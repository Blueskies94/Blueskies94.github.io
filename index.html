Previous Projects

I've had the chance to create or be a part of strategy in the projects that I've been involved. The most recent being the data migration of multiple customer interfaces into Salesforce. This was accomplished using Microsoft SQL to backup the database from a cloud server and querying the interface to export into a spreadsheet for cleaning criteria. This was loaded into Salesforce under our organization's cloud.

My first tenure as a data analyst was an internship with a political campaign. While there, I constructed a strategy for canvassing using Excel and provided a form template created in Access so that voter responses would be recorded to manage data more efficently. The result is saved time and reduced dependence on outside resources.
